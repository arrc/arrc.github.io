Features
  • Users can create account.
  • Save bookmarks in cloud.
  • Save bookmarks using either browser extension (Bloi remote) or web form.
  • Create topics to organize bookmarks.
  • Make any topic public/private.
  • Public topics and its bookmarks are visibile to the world on users profile page.
  • Use flags to indicate the importance of the bookmarks.
  • Search own bookmarks/topics.
